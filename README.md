# Minesweeper-Mac-Edition
<h3> Controls </h3>
<ul>
<li>Left-Mouse-Click - To uncover the square</li>
<li>Right-Mouse-Click - To place a flag on the square</li>
</ul>
<h3> Technologies </h3>
<ul>
<li>Python 3</li>
<li>Tkinter</li>
</ul>
